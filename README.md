# xmr-wow

Atomic swaps between Monero (XMR) and Wownero (WOW) with optional coordination
via a customized p2pool-style sharechain. WOW locks first.

## Status

Pre-release. No independent third-party audit has been performed.

## Getting Started

See [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md) for operator instructions (node setup,
swap execution, network configuration).
