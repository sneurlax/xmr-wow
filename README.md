# xmr-wow

Atomic swaps between Monero (XMR) and Wownero (WOW) with decentralized coordination
via a sharechain network. WOW-first lock order enforces atomicity without trusted
third parties.

## Status

Pre-release. No independent third-party audit has been performed.

See [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md) for operator instructions (node setup,
swap execution, network configuration).
